  • ¿Que es Arduino y para que lo utilizamos?

    Arduino es una plataforma de creación de electrónica de código abierto, la cual está basada en hardware y software libre, flexible y fácil de utilizar para los creadores y desarrolladores. Esta plataforma permite crear diferentes tipos de microordenadores de una sola placa a los que la comunidad de creadores puede darles diferentes tipos de uso.

    Arduino para nosotros es el núcleo de todo el proyecto. Esta herramienta ha sido usada con el fin de que nuestro coche tenga las funciones básicas como moverse, poder ser controlado a través de una aplicacíon móvil o un mando. Esta herramienta de Arduino también puede realizar operaciones enlazadas con otras herramientas. El coche esta programado para que además de proporcionarnos sus funciones básicas, el radar que detecte los objetos que esten en su camino.

    -Y la segunda esque el cuando conectamos el bluetooth al coche/robot lo podamos manejar.

  • Este proyecto es algo diferente ya que consiste en unir dos pequeños proyectos y convertirlos en 1 solo:

    La primera parte del proyecto es un coche arduino. Es la parte más vital del proyecto ya que la gracia de este es poder ser controlado por nosotros mismo, y sin esta parte física sería imposible. Esta parte consiste en montar y programar el arduino para que el coche se pueda teledirigir a través de una aplicación web o un mando que ya venía con el mismo coche. También debe ser programado para que pueda detectar y esquivar objetos gracias a su sensor ultrasonido.

    Este “coche/robot” le hemos puesto como MENTE X ya que nos parecio interresante el nombre. Decidimos ponerle este nombre porque a la hora de buscar ideas para la parte del radar y del coche, buscando en youtube ideas, se podía ver que toda esta clase de vehículos llevaban implementado un radar, igual que el que en nuestro proyecto.
